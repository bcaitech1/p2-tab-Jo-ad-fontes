# pstage_02_image_classification

## Getting Started    
python3 inference.py --MODEL 'lgbm' --FILE_NAME 'v2.0'

### 
evaluation.py
---
결과를 평가할 수 있는 함수가 있다

features.py
---
학습을 위한 feature를 만드는 공간

inference.py
---
데이터를 불러와 학습을 진행시키고 결과를 저장하는 공간

model.py
---
모델과 모델파라미터가 정의되어 있는 공간

utils.py
---
seed 고정 함수 또는 학습 중간중간 확인을 도와주는 함수
